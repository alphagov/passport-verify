  • createStrategy(verifyServiceProviderHost: string, logger: Logger, createUser: function, verifyUser: function, saveRequestId: function, loadRequestId: function): PassportVerifyStrategy
    • Defined in passport-verify-strategy.ts:138

    Creates an instance of PassportVerifyStrategy

    Parameters

    • verifyServiceProviderHost: string

      The URL that the Verify Service Provider is running on (e.g. http://localhost:50400)

    • logger: Logger

      A logger for the strategy. If you don't want the strategy to log you can pass an object with no-operation methods.

    • createUser: function

      A callback that will be invoked when a response with a new user is received. The user object will contain the users' attributes (i.e. firstName, surname etc.). Your callback should store details of the user in your datastore and return an object representing the user.

        • (user: TranslatedResponseBody): any | false

        • Parameters

          • user: TranslatedResponseBody

          Returns any | false

    • verifyUser: function

      A callback that will be invoked when a response with a matched user is received. Your callback should look the user up in your datastore using their pid (persistent identitfier) and return an object representing the user.

        • (user: TranslatedResponseBody): any | false

        • Parameters

          • user: TranslatedResponseBody

          Returns any | false

    • saveRequestId: function

      A callback that will be invoked to save the requestId that has been generated by the verify service provider. Your callback should save the request Id in a secure manner so that it can be matched against the corresponding SAML response.

        • (requestId: string, request: express.Request): void

        • Parameters

          • requestId: string
          • request: express.Request

          Returns void

    • loadRequestId: function

      A callback that will be invoked to load the requestId that has been securely saved for the user's session.

        • (request: express.Request): string

        • Parameters

          • request: express.Request

          Returns string

    Returns PassportVerifyStrategy

    A strategy to be registered in passport with

    passport.use(passportVerifyStrategy)
